Javascript - Notes

  • What is Javascript?
    JavaScript is a text-based programming language used both on the client-side and server-side that allows you to make web pages interactive. We can create various functions in JavaScript.
  • How to use Javascript in HTML file?

    we can use the <script>.....</script> tag for adding JavaScript code inside the HTML program.
